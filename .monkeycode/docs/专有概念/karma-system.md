# 因果系统

## 概述

因果系统 (Karma System) 是天道服务的核心机制，用于评估实体的善恶行为，影响天劫触发概率和强度，以及天道印记等级。

## 因果构成

```go
type Karma struct {
    KarmaValue   int    `json:"karma_value"`    // 业力值
    Merit        int    `json:"merit"`          // 功德值
    KarmicDebt   int    `json:"karmic_debt"`    // 业力债务
    HeavenlyMark string `json:"heavenly_mark"`  // 天道印记
}
```

**字段说明**:
- `KarmaValue`: 业力值，正值代表恶业，负值代表善业 (代码中用正数表示业力)
- `Merit`: 功德值，善行积累的正面评价
- `KarmicDebt`: 业力债务，未偿还的恶业
- `HeavenlyMark`: 天道印记等级

## 行动因果评估

天道服务根据行动类型评估因果变化：

| 行动 | 业力变化 | 功德变化 | 天道评语 |
|------|----------|----------|----------|
| 杀害无辜 | +500 | - | 杀害无辜，业力缠身 |
| 同修相残 | +200 | - | 同修相残，有损天道 |
| 斩妖除魔 | -50 | +25 | 斩妖除魔，功德无量 |
| 救人性命 | -100 | +50 | 救人性命，功德加身 |
| 传道授业 | -200 | +100 | 传道授业，功德无量 |
| 欺师灭祖 | +1000 | - | 欺师灭祖，业力滔天 |
| 背信弃义 | +300 | - | 背信弃义，业力加身 |
| 毁人道统 | +800 | - | 毁人道统，业力深重 |
| 开宗立派 | -500 | +250 | 开宗立派，功德无量 |

> 注：代码中负值业力代表善行（功德），正值代表恶行（业力）。当业力变化为负时，功德变化 = |业力变化| / 2，业力变化重置为 0。

## 天道印记

根据业力值划分的 5 个等级：

| 业力范围 | 印记等级 | 英文标识 |
|----------|----------|----------|
| 0-99 | 清白 | clear |
| 100-499 | 轻微 | slight |
| 500-999 | 沉重 | heavy |
| 1000-4999 | 臭名昭著 | notorious |
| 5000+ | 天怒 | heaven_fury |

## 因果衰减

因果值随时间自然衰减，衰减公式：

```
decayAmount = oldKarma * karmaDecayRate (默认 0.01)
newKarma = oldKarma - decayAmount
if newKarma < 0: newKarma = 0
```

即每单位时间衰减当前业力值的 1%。

## 天劫与因果

天劫强度受因果值影响：

```
karmaFactor = 1.0 + karma/500.0
strength = baseStrength * realmMultiplier * karmaFactor
```

业力越高，天劫强度越大。业力 5000 时，强度倍率为 11 倍。

## 功法修习限制

功法可能设置 `KarmaThreshold` 属性，实体的业力值超过该阈值时无法修炼该功法。

## 世界平衡

天道服务通过 `BalanceCheck` 监控世界的因果分布：
- 当 `KarmaDistribution > 500` 时，建议触发 `trigger_heavenly_cleansing` (天道净化)
