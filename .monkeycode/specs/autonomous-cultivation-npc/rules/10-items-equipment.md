# ==========================================
# 物品与装备规则 (Items & Equipment)
# ==========================================

## 1. 物品品质规则

### 1.1 品质等级
| 品质 | 属性倍率 | 获取方式 |
|------|----------|----------|
| 凡品 | 1.0x | 基础制作/掉落 |
| 良品 | 1.2x | 熟练制作 |
| 精品 | 1.5x | 高成功率制作 |
| 极品 | 2.0x | 完美制作/奇遇 |
| 仙品 | 3.0x | 天道奖励/古传承 |

### 1.2 耐久度规则
- 装备有耐久度，使用会降低
- 耐久归零后属性失效
- 可通过修理恢复（消耗材料）
- 法宝类物品耐久恢复更慢

## 2. 装备栏规则

### 2.1 装备槽位
| 槽位 | 可装备类型 | 数量 |
|------|------------|------|
| 武器 | 剑/刀/枪/杖/法等 | 1 |
| 头盔 | 各类头盔 | 1 |
| 护甲 | 衣/铠/袍等 | 1 |
| 靴子 | 鞋/履等 | 1 |
| 戒指 | 戒指类 | 2 |
| 项链 | 项链/吊坠 | 1 |
| 法宝 | 主动法宝 | 3 |
| 饰品 | 特殊饰品 | 2 |

### 2.2 装备限制
- 境界要求：装备有最低境界限制
- 属性要求：部分装备需要特定灵根
- 阵营要求：正/魔装备不可混穿（混穿触发 debuff）

## 3. 丹药使用规则

### 3.1 丹药效果
| 类型 | 效果 | 持续时间 |
|------|------|----------|
| 修炼丹 | 修炼速率 +X% | 1-24 小时 |
| 突破丹 | 突破成功率 +X% | 立即 |
| 疗伤丹 | 气血回复 X | 立即/持续 |
| 解毒丹 | 清除毒素 | 立即 |
| 爆气丹 | 全属性 +X% | 1 小时，之后虚弱 |

### 3.2 丹毒规则
- 每颗丹药有丹毒值
- 累计丹毒 > 100 触发丹毒发作
- 丹毒发作：全属性 -20%，持续 24 小时
- 可通过清毒丹或时间自然消除

### 3.3 耐药性
- 同种丹药连续使用效果递减
- 递减公式：`effect = base × (0.8 ^ consecutive_uses)`
- 停药 7 天后耐药性清零

## 4. 法宝使用规则

### 4.1 认主规则
- 法宝可认主，认主后属性 +20%
- 每人最多认主 3 件法宝
- 解除认主需特殊仪式

### 4.2 法宝技能
- 法宝自带主动技能
- 技能有冷却时间
- 技能威力与法宝品阶和主人境界相关

## 5. 材料规则

### 5.1 材料品质
- 受采集区域灵气浓度影响
- 受采集者技能等级影响
- 年份影响（灵草类）

### 5.2 材料用途
| 材料类型 | 主要用途 |
|----------|----------|
| 灵草 | 炼丹 |
| 矿石 | 炼器 |
| 妖兽材料 | 炼丹/炼器/符箓 |
| 灵木 | 阵法/炼器 |
| 特殊材料 | 多种用途 |
