# ==========================================
# NPC AI 行为规则 (NPC AI Behavior)
# ==========================================

## 1. 决策优先级

### 1.1 决策层级
| 优先级 | 类型 | 处理模块 | 频率 |
|--------|------|----------|------|
| 紧急 | 战斗/逃跑/求救 | 行为树 | 实时 |
| 高 | 突破/天劫/疗伤 | 行为树 | 每 5 秒 |
| 中 | 修炼/探索/社交 | 模板匹配 | 每 30 秒 |
| 低 | 长期规划/战略 | LLM (DeepSeek) | 每 120 秒 |

### 1.2 模板匹配规则
- **匹配阈值**: ≥ 50% 相似度使用模板
- **相似度计算**: 基于情境特征向量的余弦相似度
- **模板未命中**: < 50% 时调用 LLM 生成决策，结果存入模板库

## 2. 行为树规则

### 2.1 日常行为序列
```
SELECTOR [
  SEQUENCE [
    IS_INJURED? → FLEE → HEAL
  ]
  SEQUENCE [
    IS_UNDER_ATTACK? → COMBAT → PURSUE_IF_WEAK
  ]
  SEQUENCE [
    NEED_BREAKTHROUGH? → PREPARE → ATTEMPT_BREAKTHROUGH
  ]
  SEQUENCE [
    IS_HUNGRY? → GATHER → CONSUME
  ]
  SEQUENCE [
    HAS_GOAL? → PLAN → EXECUTE
  ]
  IDLE → MEDITATE
]
```

### 2.2 战斗行为树
```
SELECTOR [
  SEQUENCE [
    IS_OUTNUMBERED? → FLEE → HIDE
  ]
  SEQUENCE [
    IS_TARGET_WEAKER? → ATTACK → LOOT
  ]
  SEQUENCE [
    IS_TARGET_STRONGER? → SURRENDER → BRIBE
  ]
  ATTACK → REEVALUATE
]
```

## 3. LLM 决策规则

### 3.1 触发条件
以下情况调用 LLM (DeepSeek):
- 模板匹配度 < 50%
- 面临重大选择（结盟/背叛/自创功法/建立宗门）
- 与其他实体首次交互
- 世界发生重大变化

### 3.2 系统提示词结构
```
角色设定:
- 姓名、境界、性格、背景
- 当前目标、所属势力
- 因果业力、气运值

世界规则:
- 完全自主修仙世界
- 行为受天道算法约束
- 所有操作需符合角色设定

决策要求:
- 从可用操作中选择一项
- 输出格式: {"action", "params", "reasoning"}
- 推理不超过 100 字
```

### 3.3 成本控制
- 日常决策：使用 `deepseek-chat` (v3)
- 复杂战略：使用 `deepseek-reasoner` (R1)
- 速率限制：600 RPM，令牌桶限流
- 降级链：主 API → 备用 API → 行为树 → 静默

## 4. NPC 人格规则

### 4.1 人格维度
| 维度 | 范围 | 影响 |
|------|------|------|
| 道德倾向 | 正 - 中 - 魔 | 行为选择倾向 |
| 野心程度 | 1-10 | 目标激进程度 |
| 风险承受 | 0-1 | 冒险行为概率 |
| 社交偏好 | 孤僻 - 外向 | 交互频率 |

### 4.2 人格与行为映射
| 人格类型 | 典型行为 |
|----------|----------|
| 正道君子 | 救助弱者，遵守门规，不偷袭 |
| 魔道狂徒 | 抢夺资源，背叛盟友，追求力量 |
| 隐世散修 | 避免冲突，专注修炼，偶尔交易 |
| 精明商人 | 频繁交易，结交人脉，规避风险 |
| 复仇者 | 追杀仇敌，不择手段，忽视其他目标 |

## 5. NPC 学习规则

### 5.1 经验积累
- 每次成功行为强化对应决策路径
- 失败行为降低该路径权重
- 模板库持续更新，命中率逐步提升

### 5.2 功法学习
- NPC 可通过交易/传承学习功法
- 自创功法需满足条件（境界/悟性/极品灵石）
- 学习新功法后评估冲突值，过高则放弃

## 6. NPC 社交规则

### 6.1 对话生成
- 使用模板库预生成对话（≥ 50% 匹配）
- 新情境调用 LLM 生成
- 对话内容符合人格和当前关系

### 6.2 关系维护
- 定期与重要关系实体交互
- 关系强度随时间衰减，需维护
- 重大事件（生死/背叛）直接影响关系
