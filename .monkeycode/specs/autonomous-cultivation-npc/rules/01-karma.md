# ==========================================
# 因果业力规则 (Karma Rules)
# ==========================================

## 1. 业力计算规则

### 1.1 基础业力值 (Base Karma)
每个行为对应基础业力值，正值为恶，负值为善：

| 行为类型 | 基础业力值 | 说明 |
|----------|------------|------|
| 杀无辜凡人 | +500 | 杀害无修为者 |
| 杀修仙者 | +200 | 杀害有修为者 |
| 背叛师门 | +300 | 攻击/出卖师父 |
| 救人一命 | -200 | 从死亡边缘救回 |
| 扶助弱小 | -50 | 帮助低境界者 |
| 偷窃 | +100 | 窃取他人物品 |
| 自创功法 | -100 | 为世界贡献新道 |
| 平息危机 | -300 | 解除世界灾难 |

### 1.2 业力计算公式
```
Δkarma = base_karma × context_multiplier × relationship_factor
```

**修正因子计算：**
- `context_multiplier = max(0.1, 1.0 - current_karma / KARMA_CAP)`
  - 业力越高，新增业力影响越小（边际递减）
- `relationship_factor`:
  - 目标为师门：2.0（背叛加重）
  - 目标为仇敌：0.5（仇杀减轻）
  - 目标为道侣：3.0（背叛极重）
  - 无关系：1.0

### 1.3 业力衰减规则
- 每小时自然衰减率：1%
- `new_karma = old_karma × exp(-0.01 × hours)`
- 功德不参与衰减

### 1.4 业力阈值表

| 业力范围 | 标记 | 天劫概率加成 | 其他影响 |
|----------|------|--------------|----------|
| 0-100 | 清白 | +0% | 无 |
| 100-500 | 微瑕 | +5% | 交易轻微排斥 |
| 500-1000 | 业重 | +20% | 交易明显排斥，正道宗门拒绝加入 |
| 1000-5000 | 恶贯 | +50% | 天道标记，资源刷新率 -20% |
| 5000+ | 天怒 | +100% | 必遭天劫，资源刷新率 -50%，所有宗门敌对 |

## 2. 天道税规则
- 基础税率：1%
- 交易额 > 10 万：税率 1.2%
- 交易额 > 100 万：税率 1.5%
- 宗门税由宗门自定，在势力范围内生效

## 3. 因果债务规则
- 接受他人恩惠不报：+50/次
- 承诺未履行：+100/次
- 债务可通过偿还或完成对应善行消除
