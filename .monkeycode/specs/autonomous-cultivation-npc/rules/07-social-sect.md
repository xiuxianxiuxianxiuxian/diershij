# ==========================================
# 社交与宗门规则 (Social & Sect)
# ==========================================

## 1. 宗门气运规则

### 1.1 宗门实力评估
```
fortune = avg_power + history + foundation + reputation + heaven_eval
```

| 组成部分 | 计算方式 | 权重 |
|----------|----------|------|
| 弟子平均实力 | `sum(combat_power) / member_count` | 30% |
| 传承历史 | `age_years × 0.01` | 15% |
| 底蕴 | `facilities_score × 0.1` | 20% |
| 声望 | `reputation / 1000` | 15% |
| 天道评价 | 综合业力/功德评估 | 20% |

### 1.2 宗门兴衰周期
| 气运值 | 阶段 | 特征 |
|--------|------|------|
| > 80 | 兴盛期 | 弟子加入率 +50%，资源产出 +30% |
| 50-80 | 稳定期 | 正常运营 |
| 20-50 | 衰退期 | 弟子叛逃率 +10%，资源产出 -20% |
| < 20 | 崩溃期 | 可能被灭门，资源被掠夺 |

## 2. 社交关系规则

### 2.1 关系类型
| 类型 | 建立条件 | 效果 |
|------|----------|------|
| 师徒 | 双方同意，境界差 ≥ 2 | 传授功法，业力共享 10% |
| 道侣 | 好感度 > 80，双方同意 | 双修加成，共享部分属性 |
| 结义 | 3 人以上，共同经历危机 | 战斗中互相支援加成 |
| 仇敌 | 一方击杀另一方亲属/师门 | 战斗伤害 +20%，见面必战 |
| 盟友 | 宗门间签订盟约 | 资源互通，共同御敌 |

### 2.2 好感度计算
```
affinity = base + interaction_bonus - conflict_penalty
```

**交互影响**:
| 行为 | 好感变化 |
|------|----------|
| 赠送礼物 | +5 ~ +20 (视礼物价值) |
| 共同战斗 | +10 |
| 传授功法 | +30 |
| 救助对方 | +25 |
| 言语冲突 | -5 |
| 抢夺资源 | -20 |
| 击杀对方 | -100 (直接变仇敌) |

## 3. 宗门管理规则

### 3.1 宗门职位
| 职位 | 权限 | 数量 |
|------|------|------|
| 宗主 | 全部权限 | 1 |
| 长老 | 招收/驱逐弟子，管理库房 | 3-5 |
| 真传弟子 | 学习高阶功法 | 10% |
| 内门弟子 | 学习普通功法 | 30% |
| 外门弟子 | 基础资源 | 60% |

### 3.2 宗门税收
- 宗主可设定税率 (0%-20%)
- 弟子交易收入的指定比例上缴宗门
- 税收用于：宗门维护、长老俸禄、公共库房

### 3.3 宗门战争
**宣战条件**:
- 宗主或 2/3 长老同意
- 仇敌宗门自动进入战争状态

**战争规则**:
- 击杀敌方弟子业力减半（战争期间）
- 胜方可掠夺败方 30% 资源
- 败方宗门可能解散

## 4. 交易规则

### 4.1 交易流程
1. 发起方提出交易提案（物品/灵石）
2. 接收方确认或还价
3. 双方确认后天道系统执行交换
4. 交易不可逆

### 4.2 天道税
| 交易额 | 税率 | 说明 |
|--------|------|------|
| < 10 万 | 1% | 基础税率 |
| 10 万 - 100 万 | 1.2% | 中等交易 |
| > 100 万 | 1.5% | 大额交易 |

- 税收不进入任何实体账户，归入世界资源池
- 用于天道系统运行和世界危机修复

## 5. 传承规则

### 5.1 功法传承
- 开源功法：任何人可学习
- 闭源功法：需创始人/宗门许可
- 血脉功法：仅限特定血脉修炼

### 5.2 遗产继承
实体死亡后：
- 50% 物品掉落原地
- 30% 归还所属宗门（如有）
- 20% 由继承人继承（徒弟/道侣优先）
- 无继承人则散入世界
