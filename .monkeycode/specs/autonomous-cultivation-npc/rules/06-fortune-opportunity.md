# ==========================================
# 气运与机缘规则 (Fortune & Opportunity)
# ==========================================

## 1. 气运值计算

### 1.1 基础公式
```
fortune = luck + merit × 0.1 - karma × 0.05 + action_bonus
```

**参数说明**:
- `luck`: 基础气运属性 (1-100)
- `merit`: 功德值 (正向加成)
- `karma`: 业力值 (负向影响)
- `action_bonus`: 近期行为修正（7 天内）
  - 救助他人：+5/次
  - 破坏机缘：-10/次
  - 完成天道任务：+20/次

### 1.2 气运等级
| 气运值 | 等级 | 机缘倍率 |
|--------|------|----------|
| < 20 | 厄运 | 0.5x |
| 20-40 | 平淡 | 0.8x |
| 40-60 | 普通 | 1.0x |
| 60-80 | 幸运 | 1.3x |
| > 80 | 天命 | 2.0x |

## 2. 机缘触发规则

### 2.1 基础触发概率
```
P = 0.01 × fortune_factor × spiritual × exploration × balance
```

**修正因子**:
- `fortune_factor = 1.0 + fortune / 100`
- `spiritual = location_spiritual_density / 100`
- `exploration`: 区域探索活跃度系数
- `balance`: 世界平衡修正 (0.5-1.5)

### 2.2 机缘类型
| 类型 | 基础概率 | 奖励 |
|------|----------|------|
| 灵草发现 | 40% | 稀有材料 |
| 前辈遗府 | 5% | 功法/法宝 |
| 功法顿悟 | 10% | 随机功法感悟 |
| 灵根觉醒 | 1% | 隐藏灵根 |
| 天道赐福 | 2% | 气运 +20 |
| 秘境入口 | 3% | 新区域解锁 |
| 贵人相助 | 15% | NPC 好感大幅提升 |
| 无事发生 | 24% | - |

### 2.3 机缘品质
基于发现者气运值：
```
quality_tier = clamp(fortune / 20, 1, 5)
```

## 3. 世界危机规则

### 3.1 危机触发条件
| 条件 | 阈值 | 触发概率 |
|------|------|----------|
| 全服平均业力 > 500 | 每 30 天检测 | 20% |
| 灵气浓度 < 20 | 实时检测 | 30% |
| 妖兽潮周期到达 | 每 3-5 年 | 100% |
| 域外天魔入侵 | 随机事件 | 5%/年 |

### 3.2 危机类型
| 类型 | 影响范围 | 持续时间 | 平息奖励 |
|------|----------|----------|----------|
| 妖兽潮 | 单区域 | 7-30 天 | 极品灵石 1-5，声望 |
| 天魔入侵 | 全服 | 30-90 天 | 极品灵石 3-10，功法 |
| 灵气枯竭 | 多区域 | 30 天 | 灵气恢复后全员加成 |
| 天道清洗 | 全服 | 15 天 | 无（纯惩罚） |

### 3.3 危机平息判定
```
P_resolution = total_contribution / required_contribution
```

- `total_contribution`: 所有参与者贡献总和
- `required_contribution`: 危机难度系数 × 参与实体数
- 未平息则世界状态恶化（灵气降低、资源减少）

## 4. 命格与命运轨迹

### 4.1 命格类型
| 命格 | 获取条件 | 效果 |
|------|----------|------|
| 凡途 | 默认 | 无特殊效果 |
| 逆天改命 | 连续 3 次突破越级成功 | 突破成功率 +20% |
| 顺天应人 | 功德 > 5000 | 天劫概率 -30% |
| 杀破狼 | 击杀 > 100 实体 | 战斗伤害 +30%，业力 +50% |
| 天机 | 发现 5 个以上秘境 | 机缘触发率 +50% |
| 孤星 | 无师徒/道侣/宗门 | 修炼速度 +20%，心境 -20% |

### 4.2 命运轨迹变更
- 每 100 年可重新评估一次命格
- 重大行为可能直接变更命格
- 命格变更伴随天道异象
