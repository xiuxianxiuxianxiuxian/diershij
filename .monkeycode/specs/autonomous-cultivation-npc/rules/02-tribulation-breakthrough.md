## 2. 突破成功率规则

### 2.1 基础成功率

| 目标境界 | 基础成功率 |
|----------|------------|
| 炼气 | 60% |
| 筑基 | 40% |
| 金丹 | 25% |
| 元婴 | 15% |
| 化神 | 10% |
| 炼虚 | 8% |
| 合体 | 5% |
| 大乘 | 3% |
| 渡劫 | 1% |

### 2.2 修正因子
```
P_success = base_rate × accumulation × method_quality × resource × mental × luck
```

- **积累因子**: `min(1.5, 1.0 + cultivation_time / required_time)`
- **功法品质**: `active_method.quality / 100`
- **资源辅助**: `1.0 + sum(consumed_items.breakthrough_bonus)`
- **心境因子**: `mental_stability / 100`
- **气运因子**: `1.0 + (luck - 50) / 200`

最终概率限制在 `[5%, 80%]` 区间。

## 3. 天劫触发规则

### 3.1 触发条件
- 每次突破尝试均可能触发天劫
- 基础概率随境界提升（炼气 10% → 渡劫 95%）

### 3.2 天劫强度计算
```
strength = base_strength × (1 + karma / 500) × (1 + recent_breakthroughs × 0.1)
```
- **业力影响**: 每 500 点业力增加 100% 强度
- **近期渡劫**: 7 天内每多一人渡劫，强度 +10%
- **功德减免**: 每 2000 功德减少 10% 概率（最低 30%）

## 4. 突破失败惩罚

| 惩罚项 | 计算方式 |
|--------|----------|
| 修为损失 | 当前境界进度的 10% |
| 冷却时间 | `24 × realm_level` 小时 |
| 心境损伤 | -20 点心境稳定度 |
| 受伤概率 | 30%（触发则气血减半） |

## 5. 衰老规则

| 剩余寿命比例 | 修炼效率惩罚 |
|--------------|--------------|
| > 50% | 无惩罚 |
| 20% - 50% | -3% ~ -9% |
| 10% - 20% | -9% ~ -14% |
| < 10% | -20% |

**寿尽处理**:
1. 若满足突破条件 → 强制触发突破尝试（九死一生）
2. 若不满足 → 身死道消，修为清零，掉落部分物品
